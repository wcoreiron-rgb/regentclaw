---
name: investigate
description: Diagnose a bug or behavior without writing code changes.
---

1. Reproduce or restate the issue with exact route/module context.
2. Identify likely code paths and inspect only relevant files first.
3. Output:
   - observed behavior
   - root-cause hypothesis
   - confidence level
   - validation plan
4. Do not implement fixes in this skill.
5. Flag security or tenant-isolation risk immediately if present.
